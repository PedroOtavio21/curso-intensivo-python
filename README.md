# Curso Intensivo de Python - Repositório de Estudos

Este diretório contém todo o meu progresso, anotações e códigos práticos desenvolvidos durante a leitura do livro **"Curso Intensivo de Python"** (Python Crash Course), de Eric Matthes. 

O objetivo deste repositório é centralizar o aprendizado dos fundamentos da linguagem e a aplicação prática em projetos reais, servindo como uma base de consulta rápida e documentação da minha evolução como desenvolvedor Python.

---

## Sobre o Livro
O livro é dividido em duas partes principais:
1.  **Fundamentos:** Conceitos básicos de programação, de variáveis a testes de código.
2.  **Projetos:** Aplicação prática envolvendo Desenvolvimento de Jogos (Pygame), Visualização de Dados (Matplotlib/Plotly) e Aplicações Web (Django).

---

## Estrutura do Diretório

A organização segue a cronologia dos capítulos para facilitar o rastreamento do conteúdo:

```text
curso-intensivo-python/
│
├── parte_1_fundamentos/           # Conceitos básicos e lógica de programação
│   ├── capitulo_01/               # Configuração de ambiente e primeiro código
│   ├── capitulo_XX/               # (Seguindo a numeração do livro)
│   │   ├── exemplos/              # Códigos digitados durante a leitura (Alguns capítulos podem não ter)
│   │   ├── exercicios/            # Desafios "Faça você mesmo" (Try It Yourself)
│   │   └── anotacoes_e_duvidas.md # Resumo do capítulo e pontos de atenção
│
├── parte_2_projetos/              # Projetos práticos de grande escala
│   ├── projeto_1_alien_invasion/  # Desenvolvimento de jogo com Pygame
│   ├── projeto_2_data_viz/        # Gráficos e análise de dados
│   └── projeto_3_learning_log/    # Aplicação web com Django
│
└── recursos_gerais/               # Cheat sheets e links úteis
```

## Tecnologias e Ferramentas

- Linguagem: Python 3.x
- Editor/IDE: VS Code
- Ambientes: venv (Ambientes virtuais para projetos específicos)
