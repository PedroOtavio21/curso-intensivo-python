# Capítulo 1 - Primeiros Passos

## Configuração do Ambiente
- **Versão:** Python 3.x (Sempre verificar com `python --version`).
- **Variáveis de Ambiente:** Essencial marcar "Add Python to PATH" no Windows para que o terminal reconheça o comando `python`.
- **Editor:** O livro sugere o **Sublime Text**, mas acaba sendo melhor por optar o VSCode ou PyCharm.

## Como Executar
1. **Via Terminal:** `python nome_do_arquivo.py`
2. **Via IDE:** Botão "Run" ou `F5`.
3. **Interpretador (REPL):** Digitar apenas `python` no terminal para testar comandos rápidos.

## Cuidados
- *Problema:* Configurar o PATH no Windows foi confuso, precisei reinstalar e marcar a caixa de seleção na primeira vez que utilizei.
